# Network Device Hardening Checklist

## Access

- Disable unused management services.
- Use SSH for management access.
- Restrict management access to approved sources.
- Require strong credentials outside demo labs.

## Logging

- Configure centralized logging.
- Set useful timestamps.
- Capture authentication events.
- Review failed login attempts.

## Operations

- Document baseline configuration.
- Back up configurations after approved changes.
- Keep software versions documented.
- Review local users on a regular schedule.
